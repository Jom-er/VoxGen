# VoxGen – Procedural Minecraft Texture Engine

VoxGen is a procedural material engine that generates original, Minecraft-style textures using algorithmic generation. It allows creators to produce copyright-safe block, item, and material textures without relying on pre-made or copyrighted assets.

---

## Features

* Procedural generation of Minecraft-style textures
* 100% original and copyright-safe output
* Supports block and material textures
* Item textures are currently unsupported and unstable
* Customizable colors, patterns, and variations
* Ideal for resource packs, addons, and Minecraft projects

---

## Use Cases

* Minecraft Bedrock & Java resource packs
* Addons and mods
* Custom Minecraft worlds and projects

---

## Why VoxGen?

Unlike traditional texture tools, VoxGen uses algorithms instead of existing images. This ensures every texture is unique, scalable, and safe to use in public projects, marketplaces, and commercial games.

---

## Output Style

* Matches Minecraft’s visual language
* Clean, tileable, Minecraft-friendly textures
* Designed to blend naturally with vanilla-style assets

---

## How to Use VoxGen

1. Go to the VoxGen link provided.
2. Type a prompt describing the texture you want to create or select one of the preset options.
3. Review the generated texture and make adjustments if needed.
4. Once you are satisfied with the result, download the texture.
5. Share your texture on other platforms if desired, but always credit VoxGen as the source.
